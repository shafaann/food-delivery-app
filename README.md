# food-delivery-app
# 🍔 Food Ordering App

## 📌 Project Overview

The **Food Ordering App** is a simple web-based application that allows users to browse food items and place orders online.
The application is built using **HTML, CSS, JavaScript, Node.js, and SQLite database**.

This project demonstrates a basic **food ordering system with frontend UI and backend data storage**.

---

## 🚀 Features

* View available food items
* Simple user interface for ordering food
* Backend server for handling requests
* SQLite database for storing food/order data
* Lightweight and fast development using **Vite**

---

## ⚙️ Installation

### 1️⃣ Clone the repository

```bash id="clone101"
git clone https://github.com/your-username/food-delivery-app.git
```

---

### 2️⃣ Navigate to the project folder

```bash id="cd101"
cd food-delivery-app
```

---

### 3️⃣ Install dependencies

```bash id="install101"
npm install
```

---

### 4️⃣ Run the application

```bash id="run101"
npm run dev
```

The application will run on:

```id="url101"
http://localhost:5173
```
